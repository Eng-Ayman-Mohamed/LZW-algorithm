# LZW-Compression

Implementation of LZW Compression and decompression in java.

it can compress all types of file ex(txt, png, pdf ...)

# How to run

```bash
  javac Main.java
```

```bash
  Java Main filepath fileFormat (txt , png , pdf ....) Integer length (1 or 2 or 4)

```

#Example

```bash
  Java Main b txt 2
```
